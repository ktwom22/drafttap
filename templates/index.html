About Draft Tap

Draft Tap is a proprietary sports analytics terminal developed in Lebanon, Maine. It leverages Python-based linear programming via the PuLP library to solve the knapsack problem for DFS. Developed by Kyle Twombley, the platform provides live-feed optimization for DraftKings NBA, MLB, and NHL slates.
